# Bitcoin Brainwallet Checker

GUI-приложение для проверки brainwallet-адресов Bitcoin через API [mempool.space](https://mempool.space).

## Описание

Программа читает текстовый файл со строками (паролями/фразами), для каждой строки:

1. Хеширует текст через SHA-256 → получает приватный ключ
2. Генерирует Bitcoin-адреса всех типов:
   - **P2PKH** (Legacy) — compressed и uncompressed
   - **P2SH-P2WPKH** (Nested SegWit)
   - **P2WPKH** (Native SegWit / bech32)
3. Проверяет каждый адрес через API mempool.space на наличие транзакций и баланса
4. Сохраняет результаты в JSON-файлы по папкам

## Структура проекта

```
├── main.py           # GUI-приложение (Tkinter)
├── processor.py      # Логика обработки файла
├── crypto_utils.py   # Криптографические функции (ключи, адреса)
├── api_client.py     # Клиент API mempool.space
├── requirements.txt  # Зависимости
└── README.md
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Режимы работы

| Режим | Описание |
|-------|----------|
| **all** | Проверка баланса и транзакций. Сохраняет все результаты в соответствующие папки |
| **txs** | Только проверка транзакций. Сохраняет адреса с транзакциями и без |
| **balance** | Только проверка баланса. Сохраняет только адреса с ненулевым балансом |

## Выходные папки

- `with_balance/` — адреса с ненулевым балансом
- `with_txs/` — адреса с транзакциями (но нулевым балансом)
- `no_txs/` — адреса без транзакций

## Зависимости

- `ecdsa` — генерация ключей на кривой secp256k1
- `base58` — кодирование Base58Check для Legacy-адресов
- `bech32` — кодирование bech32 для SegWit-адресов
- `requests` — HTTP-запросы к API
